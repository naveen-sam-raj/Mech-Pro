# 🛠️ Mechanics E-Commerce Web Application (Full Stack)

A professional **Mechanic Tools & Spare Parts E-Commerce Web Application** built using  
**React + Vite (Frontend)**, **Node.js + Express (Backend)**, and **MongoDB (Database)**.

This application allows users to browse products, add items to cart, manage quantities, and proceed to checkout.  
It also includes an **Admin Product Management system**, where products can be added dynamically.

---

## 🚀 Project Features

### 👤 User Features

- Browse all mechanic tools and spare parts
- View product image, name, and price
- Add products to cart
- Increase or decrease product quantity
- Remove products from cart
- View total cart amount
- Proceed to checkout

### 🛒 Cart Features

- Global cart state using React Context API
- Quantity increment & decrement
- Automatic total price calculation
- Professional and responsive cart UI

### 🧑‍💼 Admin Features

- Admin can add new products from UI
- Product details stored in database
- Products appear automatically on Products page

---

## 🧑‍💻 Tech Stack Used

### Frontend

- **React.js** – Component-based UI development
- **Vite** – Fast build tool and development server
- **React Router DOM** – Page routing & navigation
- **Context API** – Global state management (Cart)
- **Tailwind CSS** – Responsive and modern UI design
- **JavaScript (ES6)** – Application logic

### Backend

- **Node.js** – JavaScript runtime environment
- **Express.js** – Backend framework for APIs
- **REST API** – Communication between frontend and backend

### Database

- **MongoDB** – NoSQL database to store product and user data
- **Mongoose** – ODM for MongoDB schema modeling

### Tools

- **VS Code** – Code editor
- **Postman** – API testing
- **Git & GitHub** – Version control
